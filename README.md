# clean-norm42-tester
A 42 norm tester who print stat on norm of your project
## DOWNLOAD
To download the script in your project please use the following command

Clone the repo and chmod the script to make it executable.
```
git clone https://github.com/SeigneurLefou/clean-norm42-tester.git
chmod +x clean-norm42-tester/*.sh
```

## EXECUTE
Execute the script by using the following command
```
./clean-norm42-tester/count_norm_script.sh
```

Or by copying the script in your repo
```
cp clean-norm42-tester/count_norm_script.sh ./
./count_norm_script.sh
```
